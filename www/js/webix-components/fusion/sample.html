Webix integration with FusionCharts (fusion-chart)

By MLukman (https://github.com/MLukman)

Parameters:

config
The FusionCharts initialization config object
require
Whether to load up additional JS files or not. Default: 
{
	charts: true,
	powercharts: false,
	widgets: false,
	gantt: false,
	maps: false
}

Functions:

getChart()
Gets the FusionCharts object

Note: if you have a licensed copy of FusionCharts, simply replace the content of fusioncharts folder with your copy.
